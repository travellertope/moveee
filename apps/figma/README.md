# Figma Make Exports

This folder holds the raw code exported from Figma Make.
It is a **reference only** — not deployed anywhere.

## How to update

1. In Figma Make, click the download/export icon and save the zip.
2. Share the zip in the Claude Code chat.
3. The agent will unzip it here and update `apps/mobile/src/theme.ts`
   and any relevant screen files.
